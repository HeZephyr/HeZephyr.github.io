# 好用的工具

## tunnelmole
[source](https://tunnelmole.com/docs/)
Tunnelmole 是一个简单的工具，可以为本地运行的 HTTP(s) 服务器提供公共 URL。例如，您可以获得一个公共 URL：
* 网络服务器
* Docker容器
* API
* 静态网站
* etc..

安装方法如下：
```shell
curl -O https://install.tunnelmole.com/n3d5g/install &amp;&amp; sudo bash install
```
使用方法：
```shell
tmole &lt;your_server_port&gt;
```
